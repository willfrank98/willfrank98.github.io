Logic Quest

Github

Logic Quest is an educational game designed to teach the user about the function of logic gates (AND, XOR, etc.). The game was created in a group of 7 as our final project for CS 3505, Software Practices II.

Being an immensely collaborative process everyone contributed to nearly every facet of the game, however I was primarily responsible for designing and implementing the UI within each level, as well as level switching and the level loading mechanism.

The game uses an intuitive drag and drop interface within each level, with an automatically updating display of the output for the given inputs. Additionally, our game was written in a way that each level was encoded in plaintext in such a way that anyone could modify the levels to whatever they desire.

This project was invaluable for teaching me about working in a larger group, as well as planning for and organizing what occasionally felt like too many people.

QtSE

Github

QtSE is a sprite editor written in C++/Qt. It's many features included multiple frames, live animation preview, GIF exporting, sprite sheet exporting, PNG exporting, textfile-based saving and loading, feathering brush, mirror paint brush, bucket fill, color replace, ellipse tool, square tool, line tool, and selection moving.

QtSE was created in a group of 6 students. I was primarily responsible for implementing textfile-based saving and loading, as well as vertically mirrored painting, and the color replace brush.

Market Predictions

Github

A class project for which the only requirement was that a major component involve machine learning. I decided to attempt to predict stock market movements using primarily news articles. The project is coded in Python using the Anaconda distribution. The final pipeline involves converting New York Times headlines for one day into term frequency-inverse document frequency representation and labeling them with the Dow Jones Industrial Average's performance over the next 10 days. Overall results were somewhat disappointing, though not discouraging. A full explanation of what was tried can be found on the github repo under Final Report. My implementation, as well as instructions for replication my results can be found on the github page as well.
